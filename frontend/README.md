# Frontend - Streamlit Web Application

User-facing web interface for the Nutrition OCR system.

## Overview
Streamlit-based web application that provides an intuitive interface for:
- Uploading nutrition label images
- Real-time OCR processing
- Viewing extracted nutrition data
- Health analysis and diet suitability checking

## Running the App

### Setup
```bash
pip install streamlit
```

### Run
```bash
streamlit run streamlit_app.py
```

The app will be available at `http://localhost:8501`

## Features
- 📸 Image upload interface
- 🔍 Real-time OCR processing
- 📊 Nutrition data display
- ❤️ Health score calculation
- 📋 Diet suitability analysis
